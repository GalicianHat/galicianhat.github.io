GimliForge

Generador de reportes ejecutivos de auditoría ofensiva impulsado por IA. Convierte hallazgos técnicos en documentos LaTeX listos para directivos.

Ver en GitHub

Estándar Red Team

Clasificación por CVSS, severidad y generación automática de métricas de riesgo.

CLI Interactivo

Asistente paso a paso en consola o ingesta automática mediante archivos JSON.

Ensamblaje LaTeX

Generación nativa de documentos .tex con gráficos dinámicos integrados.

Instalación

# 1. Clona el repositorio

git clone https://github.com/galicianhat/GimliForge.git

# 2. Instala las dependencias (Google GenAI y Matplotlib)

pip install google-genai matplotlib

# 3. Exporta tu API Key de Gemini

export GOOGLE_API_KEY="tu_api_key_aqui"

Uso

python GimliForge.py
